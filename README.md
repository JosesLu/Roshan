##Quick-Start
1. Open 2 Gitbash windows
2. Run these 2 commands in the following order:
	<br />```$ selenium-standalone start ```
	<br />```$ npm start ```
3. Open browser and navigate to ```localhost:8008```
4. Input flight details
5. Click Search button.
